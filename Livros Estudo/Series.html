Series¶

Introdução¶

A Biblioteca Pandas possui dois tipos de dados principais: Series e DataFrame. No nosso caso, iremos trabalhar com Series.

Comecemos importando as bibliotecas que iremos trabalhar.

In [1]:
## caso no tenha as bibliotecas abaixo instaladas, rode o seguinte comando

#!pip3 install pandas
#!pip3 install numpy
In [2]:
import pandas as pd
import numpy as np

Uma Series um objeto unidimensional (vetor), ou seja, uma sequncia de nmeros, por assim dizer.

Primeiro, vamos criar uma Series com o Pandas.

In [3]:
pd.Series([1, 2, 3, 4, 5])
Out[3]:
0    1
1    2
2    3
3    4
4    5
dtype: int64

Outra forma de criar atribuindo ndices aos nmeros da seguinte forma.

In [4]:
indice_alfa = pd.Series({'a':10, 'b':15, 'c':20})
indice_alfa
Out[4]:
a    10
b    15
c    20
dtype: int64

Os valores tambm podem ser criados por meio de funes externas.

In [5]:

Funcionalidades básicas¶

Assim como as demais ferramentas do Pandas, as Series possuem alguns atributos que podem ser utilizados para obter algumas informaes. Veremos alguns desses modos a seguir.

Atributo/Mtodo Descrio
serie.axes Retorna um objeto com os rtulos (labels) do eixo linha.
serie.dtype Retorna o tipo de dado da Series.
serie.empty True se o objeto Series for vazio, False caso contrrio.
serie.ndim Nmero de dimenses, por definio 1.
serie.size Nmero de elementos da Series.
serie.values Transforma os valores da Series em um array Numpy (ndarray).
serie.head() Retorna as primeiras n linhas.
serie.tail() Retorna as ltimas n linhas.

import numpy as np
In [8]:

Out[8]:

1    52.0
2    54.0
3    56.0
4    58.0
5    60.0
dtype: float64

Iterando em uma Series¶

Apesar de no ser recomendado (a prpria documentao chama de iterao preguiosa), as vezes, iterar sob uma Serie ou dataset pode ser til. Aqui vai alguns exemplos.

In [43]:
serie = pd.Series(data = [11, 20, 33, 44],
                  index = [2018, 2019, 2020, 2021],
                  name = 'Series1')

Para exibir os valores, voc pode fazer um for para exibir cada um dos valores.

In [44]:
for item in serie:
    print(item)

11
20
33
44

Faz o mesmo que o mtodo anterior, porm, ele itera retornando tuplas.

In [45]:
for item in serie.iteritems():
    print(item)

(2018, 11)
(2019, 20)
(2020, 33)
(2021, 44)

Neste caso, caso voc deseje obter o ndice e o valor em variveis separadas, mas continuar iterando, assim que se faz.

Veja que, para obter o ndice, ele ficou armazenado na primeira posio da tupla e o valor na segunda posio.

In [46]:
for item in serie.iteritems():
    idx = item[0]
    valor = item[1]
    print(idx, valor)

2018 11
2019 20
2020 33
2021 44

J nesse mtodo, os valores j so separados na prpria iterao.

In [47]:
for idx, valor in serie.iteritems():
    print(idx, valor)

2018 11
2019 20
2020 33
2021 44

Caso deseje obter apenas os ndices, voc poder usar o mtodo .keys().

In [48]:
for idx in serie.keys():
    print(idx)

2018
2019
2020
2021

Caso queira reproduzir um histograma com os dados, a maneira a se fazer especficar o tipo. Para isso, utilize o argumento "kind" com o valor "hist". O argumento bins indicar quantas divises ter no histograma, lembre-se que quanto mais "bins" tiver, maior ser preciso de processamento para renderizar o grfico.

<matplotlib.axes._subplots.AxesSubplot at 0x7fba35d56be0>
